# Third-project-customer_churn
Predicting customer churn using Decision Trees and KNN methods

Отчет.docx - Отчет по проекту, содержащий мои мысли, шаги, объяснения.
TZ_KNN_TREE - Полное ТЗ, составленное нейросетью

Структура проекта:
  1.Churn(Main) - Основная "лаборатория"
  2.MyImputerMissing - Собственный imputer для пропущщенных значений, который базируется на решающих деревьях.
  3.MyRegression -  Модуль логистической регррессии, написанный под собственные нужды и наследуемый от sklearn.LogisticRegression
  4.EDA - Модуль, содержащий инструменты для проведения исследовательского анализа.
  5.Analyze_batches - Личный препроцессинг, изначально подразумевалось, как некоторое пространство для экспериментов с выборками, но переросло в отдельный класс, для препроцессинга/масштабирования данных.
  6.Logging - Логирование проекта(Частичное), предназначалось для создания логов о работе "MyImputerMissing" и времени его выполнения
  7.Errors - Кастомные ошибки по проекту
